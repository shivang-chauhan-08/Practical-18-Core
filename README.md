# StudentCRUD

## Project Structure

- Controllers → StudentController.cs, StudentAPIController.cs
- Mapping → MappingProfile.cs
- Models/Entities → Student.cs
- Models/ViewModels → StudentCreateVM.cs, StudentUpdateVM.cs
- Models/Context → ApplicationDBContext.cs
- Repository → Generic Repository & Student Repository
- Views → Student CRUD Views (Index, Create, Edit, Details, Delete)

## Features Implemented

- MVC CRUD Operations
- REST APIs
- Entity Framework Core
- Generic Repository Pattern
- AutoMapper Mapping
- ViewModels
